The Scale

The project is about the difference between ornamentation surface and figurative massing. The three project are created by using the different numbers of the same figure. “The Tower” is a house using twelve figures to create the masing. “The Billboard” one is a mid rise with two figures and “The Box” project is a multi-funciton building with 6 figures. Each project has two parts: a concrete part and a panelized part. How the two parts divided is not reflecting the space on the inside. The figure is not only shown in the massing but also as the outline of the plans and ornamentation. All three projects have surface ornamentation, and the forms are created by the same figure. The scale is blurred with the three projet put together.
